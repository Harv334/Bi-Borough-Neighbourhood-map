# Westminster Neighbourhood Map

Interactive ward-level map for neighbourhood funding allocation across 4 Westminster neighbourhoods and 18 wards.

**Live map:** [https://YOUR-USERNAME.github.io/westminster-neighbourhood-map/](https://harv334.github.io/Bi-Borough-Neighbourhood-map/)

## What this shows

- **4 neighbourhoods:** South Westminster · West End & Marylebone · Regent Heath · St John's Wood & Maida Vale
- **18 wards** with clickable indicator data
- **229 VCSE organisations** from the Westminster One Directory (clustered)
- **GP practices, hospitals, green spaces**
- **Choropleth overlays:** crime rate, deprivation, health indicators, population

## Files in this repo

| File | Description |
|---|---|
| `index.html` | The interactive map |
| `ward_data.json` | Ward indicator data (generated by `fetch_all_data.py`) |
| `ward_geometries.json` | Ward boundary polygons for crime API queries |

## Updating the data

Data is fetched by running `fetch_all_data.py` locally (see the data pipeline repo).
Once generated, upload the new `ward_data.json` to this repository and the live map
updates automatically.

**Update frequency:**
- Crime data: monthly (police.uk)
- Claimant count: monthly (DWP Stat-Xplore — requires free API key)
- Health indicators: annually (OHID Fingertips)
- GP patient lists: annually (NHS Digital)

## Data sources

| Indicator | Source |
|---|---|
| Crime by category | [police.uk](https://data.police.uk) — ward polygon query, 3-month rolling annualised |
| UC claimant count | [DWP Stat-Xplore](https://stat-xplore.dwp.gov.uk) — monthly |
| GP registered patients | [NHS Digital ODS](https://directory.spineservices.nhs.uk) + [postcodes.io](https://postcodes.io) |
| Life expectancy, health | [OHID Fingertips](https://fingertips.phe.org.uk) — MSOA level |
| Resident population | [ONS Census 2021](https://www.nomisweb.co.uk) |
| VCSE organisations | Westminster One Directory (229 orgs geocoded) |
| Ward boundaries | [ONS Open Geography Portal](https://geoportal.statistics.gov.uk) — May 2024 BSC |

## Tech stack

- [Leaflet.js](https://leafletjs.com) 1.9.4 — mapping
- [Leaflet.markercluster](https://github.com/Leaflet/Leaflet.markercluster) — VCSE clustering
- [OpenStreetMap](https://openstreetmap.org) — base tiles
- Vanilla JavaScript, no build step required
